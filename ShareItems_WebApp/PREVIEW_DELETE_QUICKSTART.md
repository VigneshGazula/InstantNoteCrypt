## ?? How to Use

### Preview a File
```
1. Click file type (Documents/Images/Videos/Others)
2. Find your file in the list
3. Click "??? Preview"
4. View in modal window
5. Click X or outside to close
```

### Delete a File
```
1. Find file in the list
2. Click "??? Delete"
3. Confirm deletion
4. File removed permanently
```

### Download a File
```
1. Find file in the list
2. Click "?? Download"
3. File downloads from Cloudinary
```

## ?? Important Notes

### Preview
- **Images**: Instant preview
- **Videos**: May buffer based on size
- **PDFs**: Embedded viewer
- **Text**: Formatted display
- **Others**: Download prompt

### Delete
- ?? **Permanent** - Cannot be undone
- ? **Confirmed** - Must click OK
- ? **Synced** - Deletes from Cloudinary too

### File Sizes
- Shows in KB for files < 1 MB
- Shows in MB for files ? 1 MB
- Formatted to 2 decimal places

## ?? Troubleshooting

### Preview Not Working
**Check**:
- File type supported?
- Cloudinary URL valid?
- Browser console for errors
- Try download instead

### Delete Not Working
**Check**:
- Confirmation clicked?
- Internet connection?
- Check logs for errors
- Refresh page

### Modal Won't Close
**Solutions**:
- Click X button
- Click outside modal
- Press Escape (if implemented)
- Refresh page
